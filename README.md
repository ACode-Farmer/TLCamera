# TLCamera
功能描述
1.自定义相机 拍摄照片和视频
2.切换前后摄像头、设置聚焦点、横屏拍摄
3.长按录制可以上下调整焦距
4.使用GPUImage做实时滤镜，***求大神推荐其他的做图片和视频滤镜的库***
[Demo下载](https://github.com/ACode-Farmer/TLCamera)
